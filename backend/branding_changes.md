Here's a breakdown of exactly where each Admin Site Branding field reflects on the frontend:

🏷️ Site Name (Company Name)
Page / Component	Section
Navbar	Logo alt text, Sidebar menu title header
Navbar Sidebar Footer	Copyright text (e.g. "YourBrand © 2025")
Home (Loader)	Loading screen brand name
Home (Toast)	Login success message
TrendingSlot	Hero banner ("PREMIER BETTING EXPERIENCE — YourBrand")
Footer	Brand name next to logo, description paragraph, copyright
GameProvider	"Worldwide Partnerships" subtitle
Faq	FAQ question/answer text
FeaturesSection	"Why Choose YourBrand?" heading


✏️ Tagline
Page / Component	Section
Navbar	Text shown right next to the logo (e.g. "Play · Win · Repeat")


📢 Marquee Text
Page / Component	Section
NewsTicker	Scrolling ticker bar below the hero banner (hidden if empty)


📍 Site Address
Page / Component	Section
Footer	Address line below the copyright text


🔗 Dynamic Social Links
Page / Component	Section
Navbar Sidebar	Social icons at the bottom of the slide-out menu
Footer	Social icon buttons in the bottom section
All fields are optional — if left empty, the corresponding section simply won't render (no placeholder or dummy data shown).
